# Description

A blog website consisting of a home page, a page of all posts, and a page with a contact form.

# Technologies

- ReactJS
- NextJS
- CSS
- MongoDB

# Live demo

https://next-blog-app-agradecki.vercel.app

# Run locally

Download the repository and type in terminal

```
npm install
npm run dev
```
